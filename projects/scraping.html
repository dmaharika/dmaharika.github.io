Web Scraping with Beautiful Soup

This project is a web scraping application built using Python's Beautiful Soup library. The application scrapes a website and retrieves information about products listed on the site, including their name, description, price, and image URL. The scraped data is then stored in a CSV file, which can be easily analyzed and manipulated.
Features:

  • Scrapes a website using Beautiful Soup and retrieves product information
  • Stores scraped data in a CSV file for easy analysis and manipulation
  • Retrieves the name, description, price, and image URL of each product
  • Filters results based on user-specified criteria, such as price range or product category
  • Outputs a report summarizing the results of the web scraping operation


Technologies Used:

  • Python
  • Beautiful Soup
  • Pandas
  • CSV


Skills Demonstrated:

  • Web scraping using Beautiful Soup
  • Data manipulation using Pandas
  • CSV file management
  • Python programming


Code Samples:
Here is an example of the code used to scrape the website: 

                    
                        import requests
                        from bs4 import BeautifulSoup
                        import csv

                        # URL of the website to be scraped
                        url = 'https://www.example.com/products/'

                        # Send a request to the website and retrieve the HTML content
                        response = requests.get(url)

                        # Parse the HTML content using Beautiful Soup
                        soup = BeautifulSoup(response.content, 'html.parser')

                        # Find all products on the website
                        products = soup.find_all('div', class_='product')

                        # Open a CSV file to store the scraped data
                        with open('product_data.csv', mode='w', newline='') as file:
                            writer = csv.writer(file)

                            # Write the column headers to the CSV file
                            writer.writerow(['Name', 'Description', 'Price', 'Image URL'])

                            # Loop through each product and extract its information
                            for product in products:
                                name = product.find('h2', class_='product-name').text.strip()
                                description = product.find('p', class_='product-description').text.strip()
                                price = product.find('span', class_='product-price').text.strip()
                                image_url = product.find('img')['src']

                                # Write the product information to the CSV file
                                writer.writerow([name, description, price, image_url])


This code retrieves the HTML content of the website using the requests library, parses it using Beautiful Soup, and extracts the information for each product using various HTML tags and classes. It then writes the extracted information to a CSV file using the csv library.
Conclusion:
This project demonstrates how to use Beautiful Soup to scrape a website and retrieve useful information. It also shows how to manipulate and store the scraped data for further analysis. This project would be a great addition to any portfolio for those interested in web scraping and data manipulation.
